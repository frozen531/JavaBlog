# 数组

## 一维数组
### 声明
```java
int[] a;
int a[];
```

### 初始化
> 数字默认初始化为0，对象为null
```java
int a[] = new int[4]; // 默认初始化为0，动态初始化
int a[] = new int[]{1, 2, 3, 4}; // 静态初始化
```

### 长度
```java
int size = a.length;
```

## 多维数组
### 声明
```java
int[][] a;
int[] a[];
int a[][];
```

### 初始化
```java
int[][] a = new int[3][2];
int[][] a = new int[3][];
int[][] a = new int[][]{{1, 2}, {3, 4}};
```

## 特殊
```java
int[] x, y[]; // x为一维数组，y为二维数组
```
